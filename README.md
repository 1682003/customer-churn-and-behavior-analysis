## Business Problem
A retail company is facing declining repeat purchases and increasing customer inactivity. This project analyzes customer behavior and transaction data to identify churn patterns, understand purchasing trends, and generate actionable business insights to improve customer retention.

---

## Project Objective
Perform an end-to-end data analysis workflow using Python, SQL, and Power BI to clean data, analyze customer behavior, build dashboards, and generate business insights.

---

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL (MySQL)  
- Power BI  
- Jupyter Notebook  

---

## Project Workflow
- Cleaned and analyzed customer data using Python  
- Performed business analysis using SQL queries  
- Built interactive dashboard using Power BI  
- Generated insights and recommendations

## How to Run
1. Clone repository  
2. Install libraries:  
   `pip install pandas numpy matplotlib seaborn sqlalchemy pymysql`  

3. Run notebook:  
   `Customer_Shopping_Behavior_Analysis.ipynb`  

4. Execute SQL queries in MySQL Workbench  

5. Open Power BI dashboard file
